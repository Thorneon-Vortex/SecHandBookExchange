# 校园二手书交易平台使用指南

## 一、注册与登录

### 如何注册账号？
1. 点击页面右上角的"注册"按钮
2. 填写您的学号（作为唯一账号）
3. 设置昵称和密码
4. 填写联系方式（QQ或微信）
5. 点击"注册"完成

### 如何登录？
1. 点击页面右上角的"登录"按钮
2. 输入学号和密码
3. 点击"登录"即可

## 二、发布书籍

### 如何发布一本书？
1. 登录后点击"发布书籍"
2. 填写书籍信息：ISBN、书名、作者、出版社
3. 选择书籍分类
4. 上传书籍封面图片
5. 选择发布类型：出售或赠送
6. 如果是出售，设置价格
7. 描述书籍的新旧程度
8. 点击"发布"完成

### 如何下架书籍？
1. 进入书籍详情页
2. 如果是您发布的书籍，会显示"下架"按钮
3. 点击"下架"确认即可

## 三、购买书籍

### 如何搜索书籍？
1. 在首页或书籍市场使用搜索框
2. 输入书名、作者或ISBN
3. 可以选择分类进行筛选
4. 可以按价格或时间排序

### 如何购买书籍？
1. 找到心仪的书籍
2. 点击进入详情页
3. 查看书籍信息和卖家联系方式
4. 点击"立即购买"创建订单
5. 通过联系方式与卖家约定交易

## 四、订单管理

### 如何查看我的订单？
1. 点击"我的订单"
2. 可以切换"我买的"和"我卖的"

### 订单状态说明
- 待确认：订单已创建，等待线下交易
- 已完成：买卖双方确认交易完成
- 已取消：订单被取消

### 如何确认交易完成？
1. 线下完成交易后
2. 进入订单详情
3. 点击"确认完成"
4. 双方信誉分各增加5分

## 五、信誉积分

### 什么是信誉分？
信誉分是衡量用户可信度的指标，初始为100分。

### 如何获得信誉分？
- 完成一笔交易：+5分（买卖双方各得）
- 按时履约：维持良好信誉

### 信誉分有什么用？
- 高信誉分的用户更受信任
- 可以作为交易参考
