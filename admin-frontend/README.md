# 校园二手书交易平台 - 管理后台

## 技术栈

- Vue 3
- Element Plus
- Vue Router
- Pinia
- Axios
- Vite

## 安装依赖

```bash
npm install
```

## 启动开发服务器

```bash
npm run dev
```

访问 `http://localhost:3001`

## 构建生产版本

```bash
npm run build
```

## 功能模块

- 数据概览：统计用户、书籍、订单等数据
- 用户管理：查看用户列表、禁用用户
- 书籍管理：查看书籍列表、下架书籍
- 订单管理：查看订单列表、筛选订单
